//+++++++++++++++++++++++++++
// 初めに
//+++++++++++++++++++++++++++
Gimmick Partyをダウンロードして頂きありがとうございます。
このゲームはチーム制作で作られた作品となります。
このゲームは4人用でジョイパッド4つでプレイする事を前提としてますが、
ご観覧の際にキーボード使われる事を想定して、1Pのみ動かせる仕様にしています。
ですがもし機会があれば、実際に4人でプレイして頂けると幸いです。

//+++++++++++++++++++++++++++
// 開発情報
//+++++++++++++++++++++++++++
制作期間：2022/10/24 ～ 2022/11/11(およそ3週間ほど...)

開発環境(プログラム)	: C++のみ
開発環境(デザイン)		: フォトショップ、その他Webサイト(フリー素材)

//+++++++++++++++++++++++++++
// ゲームの目的
//+++++++++++++++++++++++++++
コインを入手してスコアを稼ぎ、様々なギミックを乗り越えゴールを目指しましょう！
プレイヤー全員がゴールした時にスコアが一番大きいプレイヤーの勝利となります。

//***************************
// 操作方法(JoyPad)
//***************************
移動 : Lスティック

ジャンプ : Aボタン

パンチ : Bボタン

ポーズ画面 : STARTボタン

//***************************
// 一人用の操作方法(キーボード)
//***************************
移動 : WSADキー

ジャンプ : SPACEキー

パンチ : Vキー

ポーズ画面 : Pボタン

//-----------------
// 工夫した所
//-----------------
・ただコインを取るだけではなく、ギミックによる妨害を重視した所。
・プレイヤー同士の掛け合いを増やすべく、ジャンプとパンチを追加した所。
・プレイヤー毎に個性を付ける為、色合いや顔のパーツをカスタマイズ出来る様にした所。
・ゲームのリトライ性を増やすべく、ステージを3種類 + ランダムを実装した所。

//-----------------
// 実装に苦労した所
//-----------------
・当たり判定 <- 動くオブジェクト同士の挙動をどう誤魔化すか工夫した所
・ゴール処理 <- プレイヤーを指定の位置に動かし振り向かせて順位をソートするの一連の流れを実装する所
・タイトル名 <- モチーフは早めに決まったものの世界観等が定まらず、デザイン面が大きく停滞してしまった所
